Herencia y sobrecarga

Investigación Teórica

Clase

Una clase es una plantilla o molde para crear objetos. Define qué propiedades (datos) y métodos (funciones) tendrá un objeto. Antes de las clases, usábamos funciones constructoras, pero la clase organiza todo en un solo bloque de código.


Herencia

La herencia permite que una clase (clase hija) adopte las propiedades y métodos de otra clase (clase padre). Se usa la palabra clave extends.


Sobrecarga de métodos

La sobrescritura ocurre cuando una clase hija define un método que ya existe en la clase padre. Al hacerlo, la versión de la clase hija "reemplaza" o redefine el comportamiento para sus propias instancias.


super()

La función super() se usa dentro del constructor de una clase hija para llamar al constructor de la clase padre.
